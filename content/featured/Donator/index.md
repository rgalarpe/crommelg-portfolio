Donation App is a React-based web application that allows users to donate to different companies, track their balance, and view recent donation activity. It includes theme switching (light/dark mode), responsive design for mobile devices, and structured navigation.

The project is part of a personal learning journey focused on practicing modern web development using React and related tools.
